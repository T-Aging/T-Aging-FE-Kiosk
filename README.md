# 🖥️ TA Kiosk Frontend (React, PWA)
T-Aging 서비스의 키오스크 웹앱입니다.<br/>
사용자는 키오스크를 통해 음성 기반 대화형 주문, 순차적 메뉴 선택, 옵션 설정 및 주문을 진행할 수 있습니다.<br/>
키오스크 웹앱은 AI Agent와 실시간 상호작용하며, 주문 결과는 중앙 서버(TA Kiosk Server)로 전달됩니다.

### ✨ UX 특징
- 음성 주문 결과를 메뉴 추천 배너로 시각화
- 메뉴 → 필수 옵션 → 선택 옵션 순의 단계적 질문형 주문 UI
- 주문 최종 확인 화면을 통한 정확한 주문 확인 설계
- 주문 완료 후 주문 번호 발급 및 자동 초기 화면 복귀
